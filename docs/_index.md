KAI Scheduler is a robust, efficient, and scalable Kubernetes scheduler that optimizes GPU resource allocation for AI and machine learning workloads. This documentation is organized into the following sections:

- **[Getting Started](getting-started/_index.md)** — Install KAI Scheduler and schedule your first workloads.
- **[Concepts](concepts/_index.md)** — Understand queues, fairness, priority, and how the scheduler makes decisions.
- **[User Guide](user-guide/_index.md)** — Task-oriented guides for the scheduler's features: batch scheduling, GPU sharing, topology-aware scheduling, elastic workloads, and more.
- **[Administration](administration/_index.md)** — Operate KAI in a cluster: the operator, metrics, high availability, GitOps, and migrations.
- **[Developer Guide](developer/_index.md)** — Internals and contributor documentation, including the action/plugin frameworks and design proposals.
