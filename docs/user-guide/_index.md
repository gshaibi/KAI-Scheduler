Guides for submitting and tuning workloads with KAI Scheduler's features.

- **[Batch Scheduling](batch-scheduling/_index.md)** — Gang-schedule all pods in a group together, or not at all.
- **[GPU Sharing](gpu-sharing/_index.md)** — Share single or multiple GPUs across workloads by fraction or memory.
- **[Workload Priority](workload-priority/_index.md)** — Prioritize workloads and control preemptibility within queues.
- **[Elastic Workloads](elastic-workloads/_index.md)** — Scale workloads between minimum and maximum pod/SubGroup thresholds.
- **[Topology-Aware Scheduling](topology-aware-scheduling/_index.md)** — Optimize placement using cluster network topology.
- **[Dynamic Resource Allocation (DRA)](dynamic-resource-allocation/_index.md)** — Schedule vendor hardware via Kubernetes ResourceClaims.
- **[Time-Based Fairshare](time-based-fairshare/_index.md)** — Fair usage over time using historical queue usage.
- **[Preemption Delay](preemption-delay/_index.md)** — Give pending workloads an autoscaler waiting window before evicting others.
